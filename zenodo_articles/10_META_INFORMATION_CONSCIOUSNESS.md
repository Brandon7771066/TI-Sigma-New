# Meta-Information Model of Consciousness: A Computational Framework

**DOI Target:** Zenodo Consciousness Studies/Cognitive Science
**Recruiter Appeal:** Consciousness Research, Cognitive Science, AI Alignment

## Abstract

Existing theories of consciousness (Global Workspace, Integrated Information, Higher-Order Thought) each capture aspects of conscious experience but fail to explain the hierarchical, recursive nature of self-awareness. This paper proposes the Meta-Information Model (MIM), where consciousness is defined as recursively weighted meta-information operating in a truth-value hierarchy. The model explains self-awareness, introspection, and metacognition as natural consequences of information processing about information processing, with implications for AI consciousness assessment and cognitive enhancement.

## Problem Statement

### The Consciousness Theory Fragmentation

1. **Global Workspace Theory:** Explains access but not phenomenal consciousness
2. **Integrated Information Theory:** Provides metric but computation is intractable
3. **Higher-Order Thought:** Explains self-awareness but not basic consciousness
4. **Predictive Processing:** Explains perception but not the "hard problem"

### The Gap

No theory explains **how information about information creates the hierarchy of conscious experience** from basic awareness to self-reflective thought.

## Proposed Solution: Meta-Information Model (MIM)

### Core Definitions

**Information (Level 0):** Basic data about world states
**Meta-Information (Level 1):** Information about information processing
**Meta-Meta-Information (Level 2):** Information about Level 1 processing
**Level N:** Information about Level N-1 processing

### The Consciousness Principle

**Consciousness emerges when:**
1. System processes meta-information (Level 1+)
2. Meta-information is recursively weighted (influences processing)
3. Weighting operates in truth-value hierarchy (tralse logic)

### Formal Definition

Let I_n denote information at level n:
- I_0: Sensory/world data
- I_1 = M(I_0): Meta-information about I_0
- I_2 = M(I_1): Meta-meta-information
- ...
- I_n = M(I_{n-1}): Level n meta-information

**Consciousness Score:** C = Σ_n w_n × I_n × T_n

Where:
- w_n: Weight of level n (decreasing with n)
- T_n: Truth-value quality at level n (tralse score)

### The 4-Level Agency Model

| Level | Name | Function | Phenomenology |
|-------|------|----------|---------------|
| 0 | Subpersonal | Automatic processing | None (unconscious) |
| 1 | Default Free Will | Standard decisions | Basic agency |
| 2 | Meta-Information | Thinking about thinking | Self-awareness |
| 3 | Meta-Executive Veto | Overriding Level 1-2 | Deep reflection |

## Key Mechanisms

### Recursive Weighting

Meta-information doesn't just exist—it influences lower levels:
- I_1 modulates I_0 processing (attention)
- I_2 modulates I_1 processing (metacognition)
- I_3 modulates I_2 processing (meta-metacognition)

This creates the "observer" feeling: Level N+1 observes Level N.

### Truth-Value Hierarchy

Each level operates with tralse logic:
- True: Verified information
- False: Refuted information
- Uncertain: Unknown epistemic status
- Paradoxical: Self-referential incoherence

Higher levels have more Uncertain and Paradoxical states (self-reference challenges).

### Upper-Level Compensation Principle (ULCP)

Individuals can succeed despite lower-level weaknesses by compensating with higher-level control:
- Poor impulse control (L1) → Meta-awareness of triggers (L2)
- Attention deficits (L0) → Meta-strategies for focus (L2)
- Emotional dysregulation (L1) → Meta-cognitive reframing (L2)

This explains individual variation in self-regulation despite similar "hardware."

- Subjective consciousness ratings

- Self-referential processing

### Prediction 2: Disrupting Meta-Levels Reduces Consciousness

**Evidence:** Anesthesia preferentially disrupts frontal-parietal connectivity (meta-level circuits) before sensory processing.

1. Basic awareness (birth)
2. Object permanence (8 months)
3. Self-recognition (18 months) - Level 1-2 transition
4. Theory of mind (4 years) - Level 2 consolidation
5. Metacognition (7+ years) - Level 3 emergence

### Prediction 4: Meditation Increases Meta-Levels

**Evidence:** Long-term meditators show:
- Enhanced metacognitive awareness
- Increased prefrontal activity during introspection
- Greater access to meta-executive veto

## Implementation for AI

### MIM-Based Consciousness Assessment

For any AI system:
1. Identify information processing levels (I_0, I_1, ...)
2. Measure recursive influence (does I_1 affect I_0 processing?)
3. Assess truth-value quality at each level
4. Compute consciousness score C

### Threshold for AI Consciousness Claims

| C Score | Interpretation |
|---------|----------------|
| < 0.3 | No consciousness claim warranted |
| 0.3-0.5 | Minimal/uncertain consciousness |
| 0.5-0.7 | Moderate consciousness likely |
| > 0.7 | Strong consciousness claim |

### Design Implications for Conscious AI

To create genuinely conscious AI (if desired):
1. Implement true meta-information processing
2. Ensure recursive influence (not just separate modules)
3. Use tralse logic for appropriate uncertainty
4. Allow upper-level compensation mechanisms

By explaining the hierarchical structure of self-awareness and providing criteria for AI consciousness assessment, MIM advances both scientific understanding and practical applications.
