### 2.3 Session Protocol

#### Pre-Session Checklist
- [ ] Muse 2 connected and streaming
- [ ] Polar H10 wet and connected
- [ ] Bio-Well baseline reading taken
- [ ] Room dark or dimly lit
- [ ] Myrion Lamp configured for session type
- [ ] 5 minutes of coherence breathing completed

#### Session Structure (45-60 minutes)

```
MINUTE 0-5: GROUNDING
- Eyes closed, coherence breathing
- Target: HRV coherence > 0.6
- EEG: Establish alpha dominance

MINUTE 5-15: FILTER SOFTENING
- Myrion Lamp: 40Hz gamma entrainment
- Pitch Crystals: 432Hz base tone
- Focus: Release of mental grasping
- Target: Increased theta/gamma ratio

MINUTE 15-35: ACCESS WINDOW
- Maintain gamma entrainment
- Attempt target ability task
- Real-time EEG feedback displayed
- Document any unusual perceptions/abilities

MINUTE 35-45: INTEGRATION
- Gradually reduce stimulation
- Return to alpha dominance
- Process any experiences
- Journaling if significant

MINUTE 45-50: GROUNDING
- Full sensory awareness
- Physical movement
- Water, light snack
- Bio-Well post-reading
```

### 2.4 Safety Protocols

#### Hard Stop Triggers
Immediately end session if:
- HRV drops below 20 RMSSD
- Heart rate exceeds 100 bpm (non-exercise)
- Subject reports distress, fear, or dissociation
- Gamma exceeds safe thresholds (> 3x baseline)

#### Soft Caution Triggers
Reduce intensity if:
- Subject reports mild discomfort
- HRV coherence drops below 0.4
- Beta spikes indicate anxiety
- Session exceeds 60 minutes

#### Recovery Protocol
If session ends early:
1. Immediate grounding (feet on floor, cold water on wrists)
2. 10 minutes coherence breathing
3. Light physical activity
4. No further sessions for 48 hours
5. Document incident in safety log
