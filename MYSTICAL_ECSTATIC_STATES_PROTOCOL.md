### 6 Dimensions of Mystical Experience

**1. Mystical (Unity, Oneness)**

**4. Ineffability**
- "Feeling that experience cannot be described in words"

**5. Paradoxicality**
- "Awareness of contradictory things being true at the same time"

**6. Sacredness**
- "Feeling that experience was sacred or holy"

**Scoring:**
```python
def score_meq30(responses):
    """
    MEQ30 questionnaire (5-point Likert: 0-4)
    30 items total, 5 per dimension
    
    Returns:
    - Total score (0-120)
    - Complete mystical: Total > 60 AND all dimensions > 60%
    """
    dimensions = {
        'mystical': sum(responses[0:5]),
        'positive_mood': sum(responses[5:10]),
        'transcendence': sum(responses[10:15]),
        'ineffability': sum(responses[15:20]),
        'paradoxicality': sum(responses[20:25]),
        'sacredness': sum(responses[25:30])
    }
    
    total = sum(dimensions.values())
    
    # Complete mystical experience criteria
    all_above_60_pct = all(score >= 12 for score in dimensions.values())  # 12/20 = 60%
    
    complete_mystical = (total > 60) and all_above_60_pct
    
    return {
        'total': total,
        'dimensions': dimensions,
        'complete_mystical': complete_mystical
    }
```
