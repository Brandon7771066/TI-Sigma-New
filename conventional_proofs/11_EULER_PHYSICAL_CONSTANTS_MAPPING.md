**Physical Manifestations:**
- Radioactive decay: N(t) = N₀e^(-λt)
- Population growth: P(t) = P₀e^(rt)
- Thermodynamic distributions: P(E) ∝ e^(-E/kT)

### 2.2 The Imaginary Unit (i = √-1)

**Mathematical Property:** Rotates vectors 90° in complex plane; i² = -1

**Physical Manifestations:**
- Wave functions: ψ = Ae^(ikx - iωt)
- Electromagnetic fields: E + iB
- Quantum phases: |ψ⟩ → e^(iφ)|ψ⟩
- AC circuits: impedance Z = R + iX

**Fundamental Constant Connection:**
```
i enables phase space → i connects real and imaginary
Physical interpretation: i bridges observable (real) and potential (imaginary) states
```

### 2.3 Pi (π ≈ 3.14159)

**Mathematical Property:** Ratio of circumference to diameter; appears in all periodic phenomena

**Physical Manifestations:**
- Wave equations: sin(2πft)
- Heisenberg uncertainty: ΔxΔp ≥ ℏ/2 (where ℏ = h/2π)
- Coulomb's law: F = kq₁q₂/r² (where k involves 4π)
- Einstein field equations: 8πG terms

**Fundamental Constant Connection:**
```

It encodes cyclicity—the fact that nature repeats
```

### 2.4 Unity (1)

**Mathematical Property:** Multiplicative identity; unchanged by multiplication

**Physical Manifestations:**
- Conservation laws (symmetry)
- Normalization (∫|ψ|²dx = 1)
- Dimensionless ratios
- The "whole" that is conserved

**Fundamental Constant Connection:**
```
1 represents conservation—what remains unchanged
Energy, momentum, charge, etc. sum to conserved values
```

### 2.5 Zero (0)

**Mathematical Property:** Additive identity; origin of number line

**Physical Manifestations:**
- Vacuum state
- Ground energy (though even "zero" has quantum fluctuations)
- Balance points
- Initial conditions

**Fundamental Constant Connection:**
```
0 represents the vacuum—not empty, but ground state
The "nothing" from which "something" emerges
```
